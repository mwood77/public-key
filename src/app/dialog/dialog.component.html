Decrypting your unlockable

You'll need GnuPG to decrypt your unlockable; download and install first.

Get GnuPG

1. Click the download button to get the keyfile, @wmaw9-public-key.gpg, and place it at the same location as your unlockable.

Double-click @wmaw9-public-key.gpg to import it into your keychain While technically optional, if you choose not to import my public key you won't be able to see if the unlockable file is authenic. 

    Alternative:
    gpg --search-keys nft@wmaw.me

2. Open your terminal (shell, bash, cmd, powershell, etc) and navigate to where @wmaw9-public-key.gpg and the unlockable are located. You can accomplish this with the cd command.

3. Double-click @wmaw9-public-key.gpg file to import it into your keychain, then type the following command into your terminal from step 2. 

    gpg --output A_FILE_NAME.ext --decrypt UNLOCKABLE.sig

    == Example ==
    nft name: "dip"
    nft type: image (.PNG)
    unlockable name: "dip.sig"

    gpg --output dip.png --decrypt dip.sig

4. After decryption you should see the following message in your terminal 

    > gpg --output dip.png --decrypt dip.sig
    gpg: Signature made wo  5 may 10:50:27 2021 CEST
    gpg:                using RSA key 0852C02A0846BB3020E18B0CEF3AB3E2F4581E15
    gpg: Good signature from "@wmaw9 (nft_key) nft@wmaw.me" [unknown]
