Decrypting your unlockable

You'll need GnuPG to decrypt your unlockable; download and install first.

Get GnuPG

Depending on what installer you chose, you may get a user interface ("app"). I'll only cover command line decryption here, but most "apps" will only need you to double click the unlockable to access its content.

1. Click the download button to get the keyfile, @wmaw-public-key.gpg, and place it at the same location as your unlockable.

If you're familiar with GPG, you can acquire it via the command line. 

    gpg --search-keys nft@mwood.dev

2. Open your terminal (shell, bash, cmd, powershell, etc) and navigate to where @wmaw-public-key.gpg and the unlockable are located. Most terminals accept the cd command to change directories.

3. We need to import my public key into your keychain. Type the following into the same terminal from step 2: 

    gpg --import @wmaw-public-key.gpg

If you downloaded an "app," you can probably double-click the @wmaw-public-key.gpg file to import it.

4. To decrypt the unlockable, enter the following into your terminal from step 2: 

    Take the following:
        NFT Name: "dip"
        NFT Type: png image (.png extension)
        Unlockable Name: "dip.sig"

    and fill in the [[ ]] parts of this command:
        gpg --output [[MAKE_UP_A_NAME]] --decrypt [[UNLOCKABLE_CONTENT]]

    Example:
        gpg --output dip.png --decrypt dip.sig

After decryption you should see the message beneath in your terminal, and a new file created which matches the output name you specified. 

    > gpg --output dip.png --decrypt dip.sig
    gpg: Signature made wed 5 may 10:50:27 2021 CEST
    gpg: Good signature from "@wmaw nft@mwood.dev"
