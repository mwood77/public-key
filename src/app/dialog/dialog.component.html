Decrypting your unlockable

You'll need GnuPG to decrypt your unlockable; download and install first.

Get GnuPG

Depending on what installer you chose, you may get a user interface ("app"). I'll only cover command line decryption here, but most "apps" will only need you to double click the unlockable to access its content.

1. Click the download button to get the keyfile, @wmaw9-public-key.gpg, and place it at the same location as your unlockable.

If you're familiar with GPG, you can acquire it via the command line. 

    gpg --search-keys nft@wmaw.me

2. Open your terminal (shell, bash, cmd, powershell, etc) and navigate to where @wmaw9-public-key.gpg and the unlockable are located. Most terminals accept the cd command to change directories.

3. We need to import my public key into your keychain. Type the following into your terminal from step 2: 

    gpg --import @wmaw9-public-key.gpg

If you downloaded an "app," you can probably double-click the @wmaw9-public-key.gpg file to import it.

4. To decrypt the unlockable, enter the following into your terminal from step 2: 

    gpg --output A_FILE_NAME.extension --decrypt UNLOCKABLE.sig

    ============================== Example ==============================
    nft name: "dip"
    nft type: image (.PNG)
    unlockable name: "dip.sig"

    gpg --output dip.png --decrypt dip.sig
    ============================== Example ==============================

After decryption you should see the following message in your terminal, and a new file created which matches the output name you specified. 

    > gpg --output dip.png --decrypt dip.sig
    gpg: Signature made wed 5 may 10:50:27 2021 CEST
    gpg:                using RSA key 0852C02A0846BB3020E18B0CEF3AB3E2F4581E15
    gpg: Good signature from "@wmaw9 (nft_key) nft@wmaw.me" [unknown]
